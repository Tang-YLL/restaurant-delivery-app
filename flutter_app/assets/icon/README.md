# 应用图标资源

## 图标规格

### 1. 应用主图标 (app_icon.png)
- **尺寸**: 1024x1024 像素
- **格式**: PNG（带透明背景）
- **用途**: Android应用启动器图标

### 2. 自适应图标前景 (app_icon_adaptive.png)
- **尺寸**: 1024x1024 像素
- **格式**: PNG（带透明背景）
- **安全区域**: 中心66%的区域（裁剪为圆形时可见）
- **用途**: Android 8.0+自适应图标

### 3. 启动屏幕Logo (splash_logo.png)
- **尺寸**: 512x512 像素
- **格式**: PNG（可选透明背景）
- **用途**: 应用启动时显示的Logo

## 生成图标命令

在准备好图标文件后，运行以下命令生成所有尺寸的图标：

```bash
# 生成应用图标
flutter pub run flutter_launcher_icons

# 生成启动屏幕
flutter pub run flutter_native_splash:create
```

## 临时图标说明

当前使用Flutter默认图标。请根据设计稿生成上述图标文件后替换。

## 设计建议

- 使用简洁的Logo或品牌标识
- 避免过多细节，小尺寸下应清晰可辨
- 建议使用品牌主色
- 自适应图标前景部分应该居中且不要太靠近边缘
